WordPress 7 sale el 9 de abril y cambia las reglas del juego.

Abilities API, MCP Adapter, PHP AI Client SDK — WordPress se está convirtiendo en una plataforma AI-native. Y la mayoría de los que trabajamos con WP ni siquiera sabemos que estas APIs ya existen en 6.9.

Este jueves a las 19:00 en Flowtitude vamos a revisar todo lo que ya funciona y lo que viene con WordPress 7.0:

→ Cómo un agente de IA descubre qué puede hacer tu sitio (Abilities API)
→ Cómo conectar Claude o Cursor directamente a WordPress (MCP Adapter)
→ Cómo escribir código PHP que funcione con Claude, GPT y Gemini (AI Client SDK)

Y como muestra de lo que se puede hacer con esto, vamos a regalar a los asistentes WP AI Admin — una app open-source que hemos construido esta semana para gestionar WordPress con lenguaje natural.

Un chat conectado a WP-CLI via Claude, con 29 tools predefinidas, ejecución de cualquier comando WP-CLI, 20 workflows automatizados y las 13 skills oficiales de WordPress/agent-skills cargadas por contexto.

En vez de abrir el terminal y recordar flags, le dices "haz un health check" o "actualiza todos los plugins con backup previo" y se hace.

📎 https://github.com/sabiertas/wp-ai-admin

Sesión abierta, gratuita, 90 minutos técnicos.

Para la sesión de este jueves hemos preparado algo especial.

Además de repasar todo lo que viene con WordPress 7 (Abilities API, MCP Adapter, AI Client SDK, Gutenberg Phase 3...), vamos a regalar a todos los asistentes una herramienta que hemos construido esta semana:

**WP AI Admin** — una app local para gestionar cualquier sitio WordPress usando lenguaje natural.

¿Qué hace?

🔸 Le escribes "haz un health check" → ejecuta los comandos WP-CLI y te muestra el resultado
🔸 29 herramientas predefinidas + ejecución libre de cualquier comando WP-CLI
🔸 20 workflows listos: auditoría de seguridad, review de rendimiento, limpieza de DB, migración de dominio, backup completo...
🔸 Las 13 skills oficiales de WordPress/agent-skills se cargan automáticamente según el contexto
🔸 Conecta con sitios locales (MAMP) o remotos (SSH)
🔸 Se instala con un solo comando

Es open-source y el repo ya está público:
👉 https://github.com/sabiertas/wp-ai-admin

```
git clone https://github.com/sabiertas/wp-ai-admin.git
cd wp-ai-admin && bash setup.sh
npm start
```

En la sesión del jueves haremos una demo en vivo y explicaremos cómo funciona por dentro (Claude API + tool_use + WP-CLI).
